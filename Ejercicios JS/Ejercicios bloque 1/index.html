Ejercicios del primer bloque del curso de JavaScript

1- Hacer un programa que pida dos números, que diga cual es mayor, cual es menor o si son iguales
PLUS: Si los números son menores a cero, o son NaN, los vuelva a pedir (listo)
2- Usando un bucle, mostrar la media y la suma de los números introducidos por el usuario hasta que se meta un número negativo y se mostraría el resultado (Listo)
3- Hacer un programa que muestre todos los números que hay entre dos números introducidos por el usuario (Listo)
4- Mostrar todos los números impares que estén entre 2 números introducidos por teclado(Listo)
5- Programa que muestre todos los números divisores de un número introducido en un prompt (Listo)
6- Hacer un programa que diga si un número es par o impar, debe tener su ventana prompt y en caso de no escribir un numero, que lo pida de nuevo
7- Hacer la tabla de multiplicar de un número introducido por pantalla.
